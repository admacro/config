Emacs: Xah Fly Keys Tutorial

By Xah Lee. Date: 2018-03-30. Last updated: 2018-04-03.

Here's a xah-fly-keys tutorial for beginners.

Master each step by pressing the keys.

Follow the sections here in order.

I assume you have made no customization.

On this page, DVORAK key notation is used.

First, here's layout for reference.

Master Command/Insert Mode Switch

【Alt+Space】 xah-fly-command-mode-activate-no-hook to switch to command mode.

【g】 xah-fly-insert-mode-activate to switch to insert mode.

or 【Space Space】 xah-fly-insert-mode-activate to switch to insert mode.

Repeat these until you mastered them.

Cursor Movement

Switch to command mode. Then,

c h f b keys are arrows.

g p are moving by word.

Repeat these until you mastered them.

d ; move to beginning/end of line/indent/paragraph. (try holding it down)

【Space d】 beginning-of-buffer

【Space b】 end-of-buffer

Page Up or 【Ctrl+9】 page up. (scroll-down-command)

Page Down or 【Ctrl+0】 page down. (scroll-up-command)

Text Editing

. p to delete word left/right. (xah-backward-kill-word) (xah-kill-word)

. to delete 1 char to the left. (xah-delete-backward-char-or-bracket-text)

If the character is a bracket character (any of ()[]{}), the whole bracketed text will be deleted, including the bracket. To delete just the brackets, press Ctrl+g first. To delete single bracket without pair, move cursor to the left of it, then press 5.

c to delete the current text block. (text block is separated by empty lines.) (xah-delete-current-text-block)

5 (delete-char). Delete 1 character to the right.

Note: there's no key for deleting from cursor position to end of line.

, (xah-shrink-whitespaces) for deleting blank lines or white spaces. Try holding it down.

' (xah-reformat-lines) Toggle reformat current text block into 1 long line or multiple short lines.

undo

f to undo.

Ctrl+c f to redo. To redo again, just press f 1 or more times. To reverse direction, press Ctrl+c again.

I recommend not to use any undo/redo package. If you have them installed (e.g. undo tree), uninstall it.

Cut Copy Paste

q (xah-cut-line-or-region) to cut. It'll cut current line. Try holding the key.

h (xah-copy-line-or-region) Try holding the key.

k (xah-paste-or-paste-previous) When pressed twice, it'll paste previous kill-ring text.

To paste something 5 times, Ctrl+g 5 k. Or k, then do something (e.g. press Return, or Ctrl+c) then k again.

Here's more useful commands:

【Space h】 (xah-copy-all-or-region)

【Space q】 (xah-cut-all-or-region)

【Space f】 (xah-show-kill-ring) Useful when you wanted text that you copied long ago. It'll show a new buffer with all past copied text. You can just move cursor there and copy the part you want.

Sometimes you want a temp storage for some text, use:

  • 【Space f 3】 (xah-copy-to-register-1)
  • 【Space f 4】 (xah-paste-from-register-1)

Text Selection

These are quick selections. 90% of the time, they are all you need.

6 selects text block. Try holding the key down. (xah-select-block)

7 selects line. Try holding the key down. (xah-select-line)

8 select current word. If cursor is before a bracket, select the bracket and inner text. The command name is xah-extend-selection, but currently best used for just selecting word or bracket text.

9 selects inner text between brackets or quotes, "for example, string here". (xah-select-text-in-quote)

【Space a】 mark-whole-buffer

To cancel selection, press Ctrl+c

Mark

For more complex selection, use mark.

f (set-mark-command) to mark. Then, move cursor will create selection. Press again to unmark.

【Space o】 (exchange-point-and-mark) This is sometimes useful. It puts cursor to the other side of selection.

2 move cursor to previous marked position in current buffer. (xah-pop-local-mark-ring) Try pressing it a few times.

1 move cursor to previous marked position, of all buffers. (pop-global-mark)

M-x

a to call a command by name.

or 【Space Enter】 (execute-extended-command)

Search

【b】 isearch-forward. Use this to move cursor too. (You don't need other packages to jump to locations.) [see Emacs: avy ace-jump vs isearch]

【Space g】 xah-search-current-word or selection.

When in isearch:

  • left/right arrow keys ← → do search backward/forward.
  • up/down arrow keys ↑ ↓ do previous/next search item.
  • Return exits isearch and leave cursor at the current location.
  • Ctrl+c exits isearch and leave cursor at original location.

There's no key to begin backward search. Just do isearch then press left arrow key.

Find Replace

【Space p】 query-replace

【Space t p】 query-replace-regexp

I recommend Emacs: xah-find.el, Find Replace in Pure Elisp for find replace in a directory.

Open Close Save

You can use the following standard keys for now.

  • 【Ctrl+n】 new
  • 【Ctrl+o】 open
  • 【Ctrl+w】 close
  • 【Ctrl+s】 save
  • 【Ctrl+Shift+s】 save as
  • 【Ctrl+Shift+t】 open last closed

When you are comfortable with xah-fly-keys, and feel like you want to reduce using Ctrl, then learn the following.

  • 【Space c b】 new
  • 【Space c .】 open
  • 【Space t h】 close
  • 【Space s】 save
  • 【Space c ,】 save as
  • 【Space c p】 open last closed

Open Files

【Space c g】 (xah-open-file-at-cursor)

【Space c】 (xah-copy-file-path)

【Space m】 go to dired and place cursor on file name. (dired-jump)

Open in external apps (for image, pdf, music files, etc.):

  • 【Space c ,】 (xah-open-in-external-app)
  • 【Space c o】 (xah-show-in-desktop)

Bookmark related for fast open files:

  • 【Space c l】 (bookmark-set)
  • 【Space c c】 (bookmark-bmenu-list) list bookmarks
  • 【Space c p】 (xah-open-file-fast)

Open recently opened/closed files:

  • 【Space c h】 (recentf-open-files)
  • 【Space c p】 (xah-open-last-closed)
  • 【Space c f】 (xah-open-recently-closed)

Press Space c Ctrl+d to see a full list.

You don't need any extra packages to efficiently open files.

You should never need to type file path to open file, unless it's a new file you rarely work with.

All files or directories you work daily, should be bookmarked, or with open recent.

Split Buffer

3 unsplit all. (delete-other-windows)

4 split top/bottom (split-window-below)

, Switch cursor to next pane, and if no split pane, switch to next window if any. (xah-next-window-or-frame)

【Space 3】 remove current pane. (delete-window)

【Space 4】 split left/right. split-window-right

Use a workflow of 2 windows (emacs frame) side by side. Don't use workflow that have 1 fullscreen and rely on fixed split panes.

[see Programer Workflow Efficiency]

[see Emacs: Effective Windows Management]

switch buffer

【Space g】 switch-to-buffer

【Space c .】 (ibuffer) list buffers.

  • 【Ctrl+8】 xah-next-user-buffer
  • 【Ctrl+7】 xah-previous-user-buffer
  • 【Ctrl+6】 xah-next-emacs-buffer
  • 【Ctrl+5】 xah-previous-emacs-buffer

Working with Quote and Brackets

xah-fly-keys has a system to work with brackets. Brackets includes ()[]{}“” and all Unicode brackets.

These allows you to work efficiently in any programing language with lots brackets, such as C, JavaScript, especially lisp.

You don't need other packages for working brackets. If you code elisp, i also recommend Emacs: Xah Emacs Lisp Mode

Move to Brackets

m . move to left/right bracket. (xah-backward-left-bracket) (xah-forward-right-bracket)

/ (xah-goto-matching-bracket). When cursor is on a bracket, jump to the matching one. If cursor is not on a bracket, move it to parent left bracket.

Select Brackets

when cursor is on a bracket, 8 select the whole. (xah-extend-selection)

Delete Brackets

. to delete 1 char to the left. (xah-delete-backward-char-or-bracket-text) If the character is a bracket character (any of ()[]{}), the whole bracketed text will be deleted, including the bracket. To delete just the brackets, press Ctrl+g first. To delete single bracket without pair, move cursor to the left of it, then press 5.

Change Brackets

Alt+q xah-change-bracket-pairs

Insert Brackets

Always insert left/right brackets together.

Turn on electric-pair-mode on

(electric-pair-mode 1)

Alternatively,

【Space . f】 insert parenthesis pair. (xah-insert-paren)

【Space . b】 (xah-insert-square-bracket)

【Space . h】 (xah-insert-brace)

【Space . g】 (xah-insert-ascii-double-quote)

【Space . c】 (xah-insert-ascii-single-quote)

Now, type 【Space . Ctrl+d】 to see all other brackets and quotes you can insert.

Font Size

  • 【Ctrl++】 text-scale-increase
  • 【Ctrl+-】 text-scale-decrease
  • 【Space b a】 text-scale-increase
  • 【Space b a -】 text-scale-decrease

More Advanced

ok, the above are the basics.

Now, there are about 200 commands with leader keys. These basically replace what emacs C-x does.

You might want to learn them as you go.

Remember to use Alt+q describe-function and Alt+q describe-key to find what's the key or command name.
